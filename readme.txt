== Description ==
This widget displays the saint of the day, according to the timetable of the ordinary and/or the extraordinary form of Roman Catholic rite.

= Custom =
Once the plugin Saint du jour has been activated and the widget published, it immediately displays the saint of the day with default parameters.
After, you can modify all parameters in widget's administration.
For good styling, you have to define <b>.saintdujour_forme</b> and <b>.saintdujour_saint</b> classes.

== Installation ==
= Install =
Upload "saint-du-jour" directory in wp-content/plugins/ .
Then just activate it on your plugin management page and publish the widget on your widget management page.
That's it, you're done !

= Update =
* Do the automatic update
* OR
* Deactivate Saint du jour plugin.
* Override "saint-du-jour" directory in wp-content/plugins/
* Re-activate it on your plugin management page
